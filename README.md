# Spaghetti Word

<img align="right" src="logo.png" width="150px"/>

A comprehensive suite of productivity tools for document and media preparation, built with C# and Windows Forms.

The toolkit includes automation utilities for post‑PDF OCR processing, subtitle format converters, and a custom font‑collection tool for Word documents inspired by Adobe InDesign’s packaging workflow.

Provides global formatting and cleanup routines that drastically reduce time spent on repetitive tasks, enabling teams to focus on higher‑value work and boosting overall productivity.
